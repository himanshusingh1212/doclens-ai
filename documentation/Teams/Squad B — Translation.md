# Squad B — Translation

> **Focus Area:** AI Engine Integration and Translation Management  
> **Team Size:** 13 Members (1 Lead, 12 Engineers)  
> **Primary Pipeline:** [[Translation Pipeline]]

---

## Purpose

**Squad B** handles the translation process. They integrate AI models, manage translation memory and terminology lists, and handle streaming data delivery to the UI.

---

## Key Responsibilities

- **AI Model Integration:** Manages connections to AI services (e.g., OpenRouter) and custom prompt configurations.
- **Translation Workflows:** Implements sequential page memory translation and custom output adjustments.
- **Terminology Management:** Integrates custom glossaries and terminology rules into the translation pipeline.
- **Quality Checks:** Runs translation quality verification checks (BLEU/COMET).
