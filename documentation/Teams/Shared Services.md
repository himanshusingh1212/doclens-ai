# Shared Services

> **Focus Area:** DevOps, API, Security, and Quality Assurance  
> **Team Size:** 9 Members (1 Lead, 8 Engineers)  
> **Coverage:** Cross-Squad Support

---

## Purpose

**Shared Services** provides infrastructure, security, API routing, QA, and deployment support to keep delivery pipelines running smoothly across all squads.

---

## Key Responsibilities

- **DevOps:** Manages build setups, local development environments, hosting services, and deployment pipelines.
- **API Support:** Maintains proxy servers, validates API keys, and manages routing configurations.
- **Security:** Reviews data privacy configurations, key storage, and overall application security.
- **QA Testing:** Runs end-to-end integration testing and automated regression tests.
