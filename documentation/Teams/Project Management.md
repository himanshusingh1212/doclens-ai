# Project Management

> **Team Role:** Agile Delivery Coordination  
> **Contact Escalation:** Level 2

---

## Purpose

The **Project Management** team coordinates sprint schedules, tracks task delivery progress, manages project risks, and runs agile ceremonies across all squads.

---

## Key Responsibilities

- **Agile Coordination:** Runs daily standups, sprint planning sessions, and retrospectives.
- **Roadmap Alignment:** Manages project schedules to ensure features align with release goals.
- **Dependency Management:** Tracks and coordinates deliverables that span multiple squads (e.g., pipeline integrations).
- **Metric Tracking:** Monitors sprint velocity, bug backlogs, and resource capacity.

---

## Communication Channels

- Hosts weekly sync meetings to align delivery schedules between Squad Leads.
- Coordinates status reporting to the [[Engineering Head]].
