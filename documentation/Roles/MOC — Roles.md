# 🎭 MOC — Roles

> Engineering roles and responsibilities across the DocLens AI project squads.

---

## Squad A Roles

- [[PDF Parsing Engineers]] — Handle digital PDF parsing and coordinate mapping.
- [[OCR Specialists]] — Extract text from scanned pages and images.
- [[Text Cleaning Engineers]] — Clean up extracted text and normalize layouts.

---

## Squad B Roles

- [[NLP Translation Engineers]] — Stream translations and configure AI prompt templates.
- [[Language QA]] — Perform translation reviews and localize UI copy.
- [[Terminology Management]] — Manage glossaries and translation memory systems.

---

## Squad C Roles

- [[TTS Engine Engineers]] — Build local WASM and browser-native voice synthesizers.
- [[Audio QA]] — Verify pronunciation, voice quality, and audio-text sync.

---

## Shared Services Roles

- [[DevOps]] — Manage local dev environments and build/release pipelines.
- [[API Integration Engineers]] — Maintain key validation features and worker proxies.
- [[Security Engineers]] — Audit data privacy, API integrations, and code packages.
- [[QA Test Engineers]] — Run automated integration testing and regression flows.
