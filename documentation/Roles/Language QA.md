# Language QA

> **Squad:** [[Squad B — Translation]]  
> **Operational Depth:** Depth 1 (Primary translation task)

---

## Purpose

**Language QA** specialists verify translation quality, check cultural nuances, and localize UI text.

---

## Responsibilities

- **Quality Validation:** Reviews AI-translated content to ensure readability, accuracy, and clear tone.
- **Localization Reviews:** Coordinates translations for the application's UI copy.
- **Review Routing:** Runs human reviews on translation pages that receive low automated confidence scores.
