# DevOps

> **Squad:** [[Shared Services]]  
> **Operational Depth:** Depth 1 (Infrastructure)

---

## Purpose

**DevOps** engineers maintain developer environments, set up local testing servers, configure build files, and manage host environments.

---

## Responsibilities

- **CI/CD Pipelines:** Builds and manages automated integration pipelines.
- **Dev Environments:** Maintains local dev configs and project setup guides.
- **Hosting Configuration:** Deploys and manages the Cloudflare Workers environments.
- **Diagnostic Widgets:** Partners with QA teams to maintain the system [[Memory Diagnostics]] dashboard.
