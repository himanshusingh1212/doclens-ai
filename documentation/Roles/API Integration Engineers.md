# API Integration Engineers

> **Squad:** [[Shared Services]]  
> **Operational Depth:** Depth 1 (API infrastructure)

---

## Purpose

**API Integration Engineers** maintain API routing proxies, secure credentials, and manage validation endpoints.

---

## Responsibilities

- **Proxy Maintenance:** Maintains worker scripts that route requests securely to OpenRouter.
- **Validation Endpoints:** Implements verification checks inside [[ApiKeyModal]] routes.
- **Error Mapping:** Converts API error payloads into clear warnings for users.
