# Text Cleaning Engineers

> **Squad:** [[Squad A — PDF Extraction]]  
> **Operational Depth:** Depth 2 (Clean formatting tasks)

---

## Purpose

**Text Cleaning Engineers** clean and normalize extracted text blocks, removing document noise and formatting issues before translation.

---

## Responsibilities

- **Structural Normalization:** Removes page headers, footers, repeat titles, and page numbering noise.
- **Text Correction:** Fixes common PDF extraction bugs like broken hyphenations, ligatures, and encoding errors.
- **Formatting Cleanup:** Merges split text lines and structures paragraphs correctly.
- **Content Handoff:** Formats page data into the `extraction_output.json` payload for translation.
