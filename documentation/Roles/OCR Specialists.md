# OCR Specialists

> **Squad:** [[Squad A — PDF Extraction]]  
> **Operational Depth:** Depth 1 (Primary extraction task)

---

## Purpose

**OCR Specialists** convert image-based and low-confidence pages into searchable, clean text layouts.

---

## Responsibilities

- **Image Pre-processing:** Enhances scanned page images (deskewing, binarization, layout cleaning) to improve OCR accuracy.
- **OCR Engine Integration:** Sets up and optimizes local OCR utilities.
- **Verification Monitoring:** Flag low-confidence page scans for layout reviews.
