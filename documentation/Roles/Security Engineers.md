# Security Engineers

> **Squad:** [[Shared Services]]  
> **Operational Depth:** Depth 1 (Data protection)

---

## Purpose

**Security Engineers** audit data privacy settings, secure API proxy connections, and verify code dependency integrity.

---

## Responsibilities

- **Data Privacy Reviews:** Confirms document data is processed locally and never leaves the client browser.
- **Secure Connections:** Audits worker scripts to ensure API keys are protected and never exposed to client logs.
- **Dependency Auditing:** Regularly scans third-party libraries for vulnerabilities.
