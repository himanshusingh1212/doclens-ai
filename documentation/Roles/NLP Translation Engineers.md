# NLP Translation Engineers

> **Squad:** [[Squad B — Translation]]  
> **Operational Depth:** Depth 1 (Primary translation task)

---

## Purpose

**NLP Translation Engineers** manage the integration of LLM endpoints, configure system prompts, and manage real-time text streaming.

---

## Responsibilities

- **Endpoint Routing:** Integrates model endpoints via [[OpenRouter API]].
- **Prompt Engineering:** Configures system prompts for translation, explanation, and summarization tasks.
- **Streaming Handlers:** Manages token streams and handles streaming text states in the UI.
- **Context Preservation:** Coordinates previous page summaries to maintain context across pages.
