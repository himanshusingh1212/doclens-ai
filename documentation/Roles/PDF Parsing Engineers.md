# PDF Parsing Engineers

> **Squad:** [[Squad A — PDF Extraction]]  
> **Operational Depth:** Depth 1 (Primary extraction task)

---

## Purpose

**PDF Parsing Engineers** extract text blocks and layout details from digital PDF documents to prepare them for downstream translation tasks.

---

## Responsibilities

- **Parsing Integration:** Coordinates text and structure extraction using [[PDF.js]].
- **Coordinate Mapping:** Maps text segment coordinates to coordinate arrays to support text selections.
- **Rendering Performance:** Builds and maintains the lazy rendering canvas views inside [[PdfViewer]].
- **Format Validation:** Validates incoming PDF structures and security configurations.
