# Audio QA

> **Squad:** [[Squad C — TTS]]  
> **Operational Depth:** Depth 1/2 (Verification tasks)

---

## Purpose

**Audio QA** engineers test synthesized audio output, verify voice quality, check pronunciation accuracy, and monitor speech sync.

---

## Responsibilities

- **Quality Auditing:** Listens for audio issues like voice clipping, static, or unexpected silence.
- **Pronunciation Checking:** Tests voice output for correct pronunciation of technical terminology and regional names.
- **Sync Verification:** Ensures highlighted UI text matches audio playback exactly.
