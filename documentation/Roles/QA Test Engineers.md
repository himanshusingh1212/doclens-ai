# QA Test Engineers

> **Squad:** [[Shared Services]]  
> **Operational Depth:** Depth 2 (Integration testing)

---

## Purpose

**QA Test Engineers** run automated integration and browser-level regression testing to ensure overall system stability.

---

## Responsibilities

- **Integration Testing:** Writes end-to-end test cases covering document ingestion, translation, and audio generation.
- **Regression Audits:** Runs regular quality checks to catch bugs early in development.
- **Performance Profiling:** Monitors canvas memory usage and storage growth to prevent browser crashes.
