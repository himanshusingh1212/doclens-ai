# Glossary

> Key terms, definitions, and abbreviations used across the DocLens AI project.

---

## Glossary of Terms

| Term / Abbreviation | Definition | Core Reference |
|---------------------|------------|----------------|
| **BLEU** | Bilingual Evaluation Understudy — an automated metric used to score translation quality | [[Translation Pipeline]] |
| **COMET** | Framework for evaluating machine translation quality using neural models | [[Translation Pipeline]] |
| **IndexedDB** | High-capacity browser database used to cache documents, translations, and voices | [[IndexedDB Storage]] |
| **OpenRouter** | API gateway routing requests to multiple LLM models | [[OpenRouter API]] |
| **OCR** | Optical Character Recognition — technology used to extract text from images and scanned pages | [[PDF Extraction Pipeline]] |
| **Piper** | Offline neural text-to-speech engine running locally via WebAssembly | [[Piper Neural TTS]] |
| **SSML** | Speech Synthesis Markup Language — XML format configuring pronunciation details | [[TTS Pipeline]] |
| **TTS** | Text-to-Speech — technology converting written text into spoken audio | [[Text-to-Speech]] |
| **WASM** | WebAssembly — binary format enabling high-performance code execution in the browser | [[Tech Stack]] |
